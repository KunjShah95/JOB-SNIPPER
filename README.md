# 🎯 Job Snipper AI - Advanced Resume Analysis Platform

A comprehensive AI-powered resume analysis and job matching platform built with Streamlit, featuring real AI capabilities, robust security, and advanced analytics.

## ✨ Features

### 🔒 Security First
- **File Validation**: Comprehensive security checks for uploaded files
- **MIME Type Verification**: Prevents malicious file uploads
- **Size Limits**: Enforced file size restrictions (10MB max)
- **Content Scanning**: Basic malware pattern detection
- **Secure File Handling**: Temporary file management with cleanup

### 🤖 Real AI Capabilities
- **Advanced NLP**: Uses spaCy and Transformers for text analysis
- **Skill Extraction**: Intelligent identification of technical skills
- **Structure Analysis**: Resume formatting and completeness scoring
- **Job Matching**: AI-powered compatibility analysis
- **Smart Recommendations**: Personalized improvement suggestions

### 📊 Analytics & Tracking
- **Performance Metrics**: Detailed analysis tracking
- **User Analytics**: Session and event monitoring
- **Score Distribution**: Statistical insights
- **Export Capabilities**: JSON/CSV data export
- **Historical Data**: Analysis history and trends

## 🚀 Quick Start

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/KunjShah95/JOB-SNIPPER.git
   cd JOB-SNIPPER
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements_new.txt
   ```

3. **Download spaCy model**
   ```bash
   python -m spacy download en_core_web_sm
   ```

4. **Run the application**
   ```bash
   streamlit run app.py
   ```
